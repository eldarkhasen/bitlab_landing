Программа курса

Основы программирования

Мы научим вас основам программирования и алгоритмизации. Вы сможете писать свои собственные алгоритмы, поймете как работать с массивами и со структурами данных.

Основы ООП

Вы изучите все 4 основные концепции объектно-ориентированного программирования, которые фундаментально необходимы для разработки приложений и прочих систем.

JDBC - База данных

Умение работать с базой данных - является одним из основных требований для создания программного обеспечения. Вы научитесь разрабатывать приложения связанные с СУБД (базой данных).

Потоки и параллельное программирование

Создадим многопотчное приложение, которое будет запускать все процессы параллельно! Это позволит нашему приложение сдерживать большие нагрузки!

Сети + сетевое программирование

Вас обучат сетевому программированию на языке Java, поймете концепцию клиента - сервера, создадите свой собственный мультиклиентный сервер с использованием потоков.

И прочие технологии для разработки

Java - это самый мощный инструмент разработки ПО! Мы научим вас мыслить как разработчик, и поставим крепкий фундамент навыков программирования.
